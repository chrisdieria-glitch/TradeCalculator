# TradingCalculator

A single-screen React Native (Expo) app that splits a trading capital into 5 risk-allocated trades.

## Allocations

| Trade   | Percentage |
| ------- | ---------- |
| Trade 1 | 5%         |
| Trade 2 | 10%        |
| Trade 3 | 20%        |
| Trade 4 | 40%        |
| Trade 5 | 25%        |

## Tech

- Expo SDK 54
- React Native (JavaScript)
- No navigation, no backend
